# FIWAtec ModCom Studio

**FIWAtec ModCom Studio** is a Windows desktop application for Modbus RTU and Modbus TCP diagnostics, bus scanning, register testing and documentation.

This release contains the ready-to-use application files for Windows.

## Main Features

* Modbus RTU scanner
* Modbus TCP support
* Safe address scan without writing
* Manual read and write functions
* Write + read verification
* Cyclic read mode
* Cyclic write mode
* Bus monitor with TX/RX telegram display
* Participant database
* Register-list database per device type
* Bus-structure import/export
* Scan-session save/load
* Found-address list
* Register autoscan after detected slave
* CSV export
* TXT, HTML, RTF/Word and PDF report export
* Modern Windows desktop interface

## Usage

Download the release package, extract it and start:

... cmd
FIWAtec_ModCom_Studio.exe

## Requirements

Windows 10 or Windows 11.

The release build is published as a self-contained Windows x64 application.
